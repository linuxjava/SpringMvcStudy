- day02_03springmvc_exception_intercepter
    - springmvc中的异常与拦截器
    - 对比学习javeweb中的过滤器和监听器
    - 拦截器与过滤器区别
         1. 过滤器是Servlet规范的一部分，任何框架都可以使用过滤器技术。
         2. 拦截器是SpringMVC框架独有的。
         3. 过滤器配置了/*，可以拦截任何资源。
         4. 拦截器只会对控制器中的方法进行拦截
